Seminarix is a Live-medium based on sidux, which is developed for the education of teachers

User orientation

Nothing is, in the literal sense of the word, sold on this Live-medium, only information about an idea is offered. Because of that it is not necessary to develop a fancy advertising design with persuasive elements.

It is enough if the user does not get scared at the beginning and more importantly that he/she finds, easily and fast, the information they are looking for.

Because of this these pages do not need to follow a widely used bad habit.

You need not tell the user which fonts he should use or the font size, you need not stimulate the user with animated graphics and you need not increase the product's cost to cover the cost of the web-site creation:

In short, you do not need to tell the user how he has to view the site.

It is always positive to let the adult user set the settings how he/she wants them.

In detail:

  1. no predefined font type
  2. no predefined absolute sizes
  3. no predefined monitor resolution

Only by following the above, are the user's settings, which he gave to the browser, fully used.

All this has to be realised in the source code and tested after the web-site's production.
