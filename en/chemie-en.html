Seminarix is a Live-medium based on sidux, which is developed for the education of teachers.

   Chemistry Section

Notes

The programs in the chemistry section are oriented towards providing information and represent the periodical system.

When the chemical processes are written in Java then they can be usually simulated platform independently.

Then they can be also used in Linux. Moreover there is a game called "KAtomic", which looks similar to the game Sokoban.

In this game atoms are created from molecules.

In all cases in which we found additional help on the Internet, we have linked it with Info under the program description.

Some links are magazine articles, some are Wiki-Articles. Since we are not the owners of these Articles, we can not guarantee that these links will work when you depend on them to get help.

At the time of writing, all information is current.

In matters of legality, what is written in the Impressum holds.

Program Introduction

The programs can be started/installed with the help of the program-starter. The programs of this page can be found under the menu item "Chemistry". You can find the program-starter in the "Kde-Menu" in the sub-menu "Utilities", or you can just click on the program-starter icon on the desktop.

Chemtool

Chemtool is a 2D-Molecule editor, in which you can compose organic molecules or import molecules in mol and pdb format, (the imported molecules can be edited. 3D-Molecules can be also imported and transformed into 2D-versions).

The created molecules can be exported in various formats. (The program possesses a help program with which you can determinate the sum formulas and calculate the mol-masses.)

Chemtool  |  Info Chemtool

 

GPeriodic

GPeriodic is a small game about the periodic table of elements which contains detailed information in a table for every element that is contained in the periodic table of elements.

GPeriodic  |  Screenshot Gperiodic (204 kb)

 

Kalzium

Kalzium is a program about the periodic table of elements, that informs you over everything that is contained in the PSE. (Moreover you can follow the development of Mendelejew's PSE using a time-line.)

It contains various interactive features, that make the numerous facts accessible for students of all ages, (among them is a quiz and the possibility to calculate molecule masses).

Info 1 Kalzium  |  Info 2 Kalzium

 

KAtomic

A game that comes from the KDE-Games and reminds us a bit of Sokoban, in which you build various atoms by combining molecules.
