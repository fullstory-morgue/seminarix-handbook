Seminarix is a Live-medium based on sidux, which is developed for the education of teachers

  General Section

Notes

In this "General" section you will find a list of programs which can be helpful with the teacher's professional duties.

There are, besides the very fast KOffice-Package, programs to play videos and multimedia files and also programs like graphic editors or the DeskTop Publishing program Scribus.

Also intergrated is a program to visualise thoughts and processes (vym). There are a lot more Applications.

For most "teacher's work", these Applications should be enough. We will introduce here the most important ones.

If you are looking for help or information for one of the KOffice packages, click on the "Help" menu in the respective program.

Reading http://docs.kde.org/development/en/koffice/ can also be helpful - there you can find all the KOffice-Manuals and information that did not make it on to the live medium due to limited space.

In all cases in where we have found additional help information on the Internet, we have linked it with Info under the program description. Some of the links are only availble in English.

Some of them are magazine articles, some are Wiki-Articles. Since we are not the owners of these Articles, we can not guarantee that these links will work when you depend on them to get help.

At the time of writing, all information is current.

In matters of legality, what is written in the Impressum holds true.

Programs Introduction

The programs can be started/installed with the help of the program-starter. The programs on this page can be found under the menu item "General". You can find the program-starter in the "Kde-Menu" in the sub-menu "Utilities", or you can just click on the program-starter icon on the desktop.

KOffice

The KDE-Project has developed a widely usable Office-package which contains a word editor, spreadsheet editor, pictures editor, formulas editor, and programs to work with scalable vector graphics, diagrams, slides presentation, a data-bank management program and a report generator.

These applications are usually more than enough for the standard educational and teaching requirements.

KOffice is a function rich office solution which is integrated into the KDE desktop.

It supports the OASIS OpenDocument file format, which is an open Standard for Office-applications. Likewise this file format is used by the current OpenOffice.org.

Moreover a large selection of filters are offered so that you can exchange documents with other popular (and proprietary) office programs like for example Microsoft Office products. (Note: The saving and opening with KOffice of proprietary Microsoft Office file formats does not always progress without some errors but the files can usually be corrected with a few manual adjustments.)

If you require a more complex set of function rich applications, you should install openoffice.org, OpenOffice.org.

(OpenOffice.org is also available for other Operating Systems)

KOffice  |  Info KOffice

 

Scribus

A frame-oriented and professionally usable Desktop-Publishing program with which you can solve special tasks in the area of page-formation.

This is particularly interesting when formation-rules of perception psychology are taken seriously in the preparation of worksheets.

It is also available for other Operating Systems.

Scribus

 

Quanta Plus

Quanta Plus is a development environment for the creation of webpages. It offers project management and supports arbitrary markuplanguages but it focuses on web-based languages and technologies like HTML, XHTML, CSS, JavaScript, Java or PHP.

Quanta Plus supports a multitude of external components and tools, which support the development. Therefore Quanta provides a rich but still easy to use environment with which you can create and manage your websites fast and with efficiency.

Quanta Plus

 

Gimp

With Gimp you have an exceptionally powerful program for editing and manipulating graphics files. With its support for multiple layers, numerous filters and a lot of possibilities to edit pictures, it can be compared to expensive and proprietary professional products.

It is also available for other Operating Systems.

Gimp  |  Info Gimp

 

Inkscape

Inkscape is a free vector-based Graphics and drawing program. Inkscape is for example, suited for the creation of logos, vector-art, technical diagrams, maps, city maps, Flyer-Layouts, CD-covers, posters and comics.

It is also available for other Operating Systems.

Info Inkscape

 

Kile

Kile is a programing environment for LaTeX. The basis is a text editor which supports Syntax-Highlighting, code completion, spell-checking and has menus to make the insertion of specific characters and symbols (for example the Greek alphabet or mathematical operators) easier.

In addition to the above, there is an assistant to help create new documents from existing templates and to insert graphics, directories or tables. After you have created the LaTeX-Document, by clicking on a button a PDF is created and opened in a PDF viewing program.

Kile  |   Info Kile

 

KImageMapEditor

With KImageMapEditor you can edit a picture in such a way so that some fixed parts of it contain for example a link. This can be useful in web projects where for example moving the mouse over a fixed part of the picture could show or follow a link.

Helpful in web projects.

KImageMapEditor

 

Kaffeine

The Kaffeine Media Player is a highly portable, open source media player not only for various audio/video codecs and file formats but also for DVDs and VCDs.

Kaffeine supports various streaming protocols.

Kaffeine - Mediaplayer - Homepage and Information the program Kaffeine may have a local language help file.

 

Audacity

Audacity is a free, open source, easy to use audio editor and recorder for Windows, Mac OS X, GNU/Linux and other operating systems.

You could use Audacity for:

  • Live Audio recording
  • Digitalising cassettes and records
  • Editing Ogg Vorbis, MP3 and Wave files
  • Cutting, copying and mixing of sounds and files
  • Changing a recording's speed and pitch

and much more! Look at the complete list.

It is also available for other Operating Systems.

Audacity

 

vym

vym is a very useful program to represent bigger and more intellectual connections and projects and with its help, you can create Mindmaps.

With the help of the right mouse button, you can add objects into the mind-map and you can edit the "thoughts card".

vym (view your mind)

 

KChart

KChart is the diagram-creating program of the KOffice-Suite. It can be used to add diagrams into KOffice-documents, for example in Kword, the KOffice's word editor, or KPresent, the KOffice's presentation program.

KChart

 

KDissert

A program from the KDE-Project with which you can create Mindmaps. Very useful for representing bigger and more intellectual connections and projects. With the help of the right mouse button, you can add objects into the mind-map and you can edit the "thoughts card".

KDissert

 

KEducaBuilder

KEducaBuilder can create and edit interactive form based tests and exams (Multiple-Choice-Tests) which can later be run with Keduca. The questions and a selection of answers (right or wrong) are choosable. Also definable is the time given to answer a question and the points each question gets. Graphics and pictures can be added to the questions.

KEducaBuilder

 

 

KEduca

With the help of this program you can create tests, quizzes and multiple choice questions with arbitrary contents and then you can run them on the computer.

KEduca

 

Kexi

Kexi is a free and open Data-bank-Manager and is developed as an alternative to MS-Access. Kexi can be used to generate Data-bank schema, to edit, add, query and manage the data of a data-bank. Forms could be create with Kexi, to get a tailor-made user environment for the data management. All data-bank objects like tables, queries and forms are held in a relational data-bank. Data can be taken from a MySQL or PostgreSQL data-bank so it can be used with Kexi and Kexi-Data-bank files can be exported into a MySQL or PostgreSQL server. Moreover Kexi can take over MS-Access-Data-bank-Tables. Macros and the programing languages Python and Ruby are supported (still in development).

Kexi

 

KPlato

KPlato is a program to plan and manage projects and it is part of the KOffice-Suite. KPlato offers resource management, a hierarchical calendar, tasks with dependencies and sub-tasks, diagram creation, planing of project's deadline (pessimistic, expected, optimistic), a project-structure-plan (PSP), a 'critical path with resources' and embedding in other KOffice-applications.

KPlato

 

KPresenter

KPresenter is a presentation program contained in KOffice. With the help of KPresenter, slides for an on-screen presentation or for printing, can be prepared. The slides can contain text and graphics in various formats and of course components from other KOffice-programs. An on-screen KPresenter presentation can be exported into HTML and so it can be directly used in websites for example as a slide-show.

KPresenter

 

KWord

KWord is an easy to use Open-source word editor and part of the KOffice Office-package which was specially developed to be used with KDE. A lot of standard functions are integrated into KWord, however KWord does not look 'functions bloated' like other word editors. For the time being, a unique function of KWord is its ability not only to print to PDF-Documents but also to read and edit them. Depending on the pattern, the transfer is more or less equivalent to the original.

KWord

KPDF

Program to view PDF files.

KPDF

 

KSociograma

KSociograma helps in creating a sociogram for a study group. At the same time various questions, to be chosen/declined from the students, can be given: Who with whom and with this but without that. The evaluation result is statistically and graphically represented.

(The 'admin' starts with the password ksocio and can add a sociogramm).

 

Krusader

A simple, easy, powerful, graphical, twin-panel file manager. It provides all the file management features you could possibly want. For example extensive archive handling, mounted file system support, FTP, advanced search module, viewer/editor, directory synchronization, file content comparisons, powerful batch renaming and much, much more. Krusader supports many archive formats (if installed): ace, arj, bzip2, deb, gzip, iso, lha, rar, rpm, tar, zip and 7-zip and handles other KIOSlaves such as smb:// or sftp://.

Manual is linked under Info.

Krusader  |  Info Krusader

 

Midnight Commander

The Midnight Commander works in the command line with the twin view technique.

Midnight Commander  |  FAQ Midnight Commander

 

XSane

A graphical front-end for the scanner program SANE. With its help the computer can interpret the information from the scanner.

A shortcut to the program Gimp is available.

XSane

 

Kooka

Kooka is a graphical interface, integrated into KDE which should help ease operating the scanner.

Kooka  |  Info Kooka

 

iTALC

iTALC (Intelligent Teaching And Learning with Computers) is a free screen control software which has mainly schools in its usage scope.

iTALC provides, besides others, the following functions:

  • Overview mode (small preview windows for all student-computers)
  • Window control or the ability to watch over specific computers in full-screen
  • show a demo (either full-screen or in a window) – the teacher's screen is transfered onto all student-computers in real-time
  • Locking of student-computers, to steer attention to other things
  • Text-messages sending
  • Snapshots to secure evidence for example when the rules of usage are violated

(iTALC is also available for other Operating Systems)

iTALC  |  Info iTALC

 

LaTeX

LaTeX is a professional text-type system for scientific work. Its strengths are writing formulas and optical quality.

Seminarix-latex installs a complete LaTeX-environment and contains numerous example files for beginners and suggestions for creating exercise sheets and letters.

LaTeX
