Seminarix is a Live-medium based on sidux, which is developed for the education of teachers

  Section Educational Games

Notes

With the use of educational games, many people learn faster the possibilities that a computer offers.

Working with young children (for example, primary school children), the chance is offered to have them work with easy routines (for example working with the mouse or the keyboard) and play various educational games.

Programs Introduction

The programs can be started/installed with the help of the program-starter. The programs of this page can be found under the menu item "Educational Games". You can find the program-starter in the "Kde-Menu" in the sub-menu "Utilities", or you can just click on the program-starter icon on the desktop.

Childsplay

Childsplay is a collection of educational games for children, similar to GCompris. Childsplay contains among others, Maths games, writing games and it can be extended with other games.

Childsplay

KTuberling

KTuberling is a stamp program which has 3 ready sets of background pictures and matching stickers. Contained sets: Aquarium, KTuberling, Tux.

The program is a part of KDE and it is contained in the KDE-Games package.

KTuberling

KTouch

KTouch is a typewriting training program for KDE.

To begin with you have to choose the right keyboard-layout in Settings - Keyboard Layouts (probably English - en). Then under File - Edit Lecture choose "Open a default lecture" and there select a lecture from the drop-down menu (probably english.ktouch.xml).

KTouch  |   Info KTouch

Pingus

Pingus is an Open-Source clone of the old game "Lemmings": Save the penguins!

Pingus

BlinKen

Your objective in BlinKen is to reproduce key or sound sequences which get longer with every round. With that it presents a special way of memory-training.

On BlinKen's game board, there are 4 differently colored buttons and each one of them has a different sound assigned to it. During the game the buttons light up in a random sequence which the user has to memorize and repeat.

If the user manages to repeat the sequence in the right order, he reaches the next game level in which he has to repeat the same sequence plus an added button. When the sequence is repeated in the wrong order, the user loses and he has to start from the beginning.

Your target is to obtain as many points as possible. Every successful round gives you one point, a successful sequence of 8 buttons will give you 8 points in the top-scorers list.

To use the sound sequences, a ready to work sound-card with speakers or headphones is needed. With the help button (symbolized by a book), the manual can be called.

BlinKen

Einstein

Einstein Puzzle is a program for people which are good in logical thinking. The objective is to fill a 6x6 field with symbols. For this, hints are given which describe the relationship between the symbols' positions.

The program contains a help-file in English.

Einstein

Gcompris

Gcompris is a large collection of educational games for children in kindergarten and primary school. With gcompris you can:

  • learn to work with the mouse and the keyboard
  • learn basic mathematical operations
  • learn to read an analog clock
  • learn how to recognise letters by their names
  • learn how to read
  • play small games (Memory, Puzzle-games, etc.)
  • Create small animations
  • and much more

Gcompris

gTans

gTans is an electronic version of the Chinese puzzle game Tangram in which the player has to arrange a set of pieces to match a given shape.

gTans

KAnagram

KAnagram is a letters puzzle game: In the game you are given a word with its letters placed randomly and you have to put the letters in the correct order to find the word.

There is no time limit and you have arbitrary many tries to solve the puzzle.

With Kanagram students get a set of vocabulary-files to play with. More can be downloaded from the Internet.

Kanagram provides, with the help-button (symbolised by a book), a manual.

KAnagram

KHangMan

KHangMan is based on the known hangman game. It is suitable for children that are 6 years old or older and it has four difficulty levels:

Animals (Animal names), easy, medium and hard. A word is randomly chosen, its letters covered and the student has to guess the word by giving one after the other letters he assumes are part of the word.

Every time you give a wrong letter, part of the hangman is drawn. Students have to guess the word before the hangman is completely drawn! For that you have 10 tries.

You can call the manual by using the help-button.

KHangMan

LMemory

LMemory is the known memory card game with covered picture-cards. Lmemory has various difficulty levels. As an extra option you can choose for example to have to find 3 card which are the same instead of 2.

LMemory

Tipptrainer

Tipptrainer is used for learning and training the touch type system.

The student is provided with numerous tutorials to learn how to type specific letter combinations and texts. There are courses for beginners and more advanced users.

Tipptrainer

TuxType

TuxtType is a training-software with which children can learn to use the keyboard in a playful manner.

The principal of the game is similar to that of the program 'Missile Command', which means that fish with letters fall from the top of the screen and Tux has to catch them and eat them. For that you have to type in the right letter/word.

There are various game-modes you can choose - from typing single letters through to long words. Three difficulty levels are offered.

TuxType

Tuxmath

Tuxmath is a mathematics trainer for children.

The principal of the game is similar to that of Missile Command. Equations fall from the sky and Tux has to give the right solution to his laser gun or else the cities will be destroyed.

TuxMath
