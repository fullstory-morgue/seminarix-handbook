  Section Mathematics

Notes

Mathematicians have naturally a close working relationship to computers and operating systems.

This is not only because the computer only understands two states, namely 0 and 1.

As a consequence there is a variety of programs in Linux for mathematicians. Below you will only find a small selection.

Programs Introduction

The programs can be started/installed with the help of the program-starter. The programs of this page can be found under the menu item "Mathematics". You can find the program-starter in the "Kde-Menu" in the sub-menu "Utilities", or you can just click on the program-starter icon on the desktop.

Dia

Dia was inspired by the proprietary program 'Visio'.

With Dia you can prepare various kinds of diagrams, flow-charts and structural drawings. Logical switches and networks are also representable. The products can be exported into various formats like EPS, SVG, XFIG, WMF and PNG.

Dia (+dia2code)  |  Examples Dia

Gnuplot

Gnuplot is a powerful command-line oriented interactive plot program which is also used by graphically based programs like wxmaxima.

The program is used to create 2D and 3D graphs of data fields and functions.

It supports numerous formats, including drivers for numerous printers, (La)TeX, (x)fig, Postscript, and more. The data fields and the self created functions can be driven with an internal C-like programing language.

Gnuplot can also work with complex numbers.

Gnuplot

KBruch

Program to learn calculating with fractions  - from the edu.kde.org Project.

KBruch is a program to learn calculating with fractions.

For training with addition/multiplication/division, comparing, converting and to factorize/shorten.

KBruch  |   Info KBruch

KCalc

KCalc is a software scientific calculator for your computer's desktop.

KCalc provides all the common calculator functions (also statistic functions). The program allows you to transform decimal, hexadecimal, octal and binary numbers into one another.

KCalc provides numerous mathematical and natural constants and allows you to define your own constants. Calculations can be copied into other applications.

KCalc

KIG

Interactive geometry under Linux. A small overview can be found by following the link "KIG". (in English)

KIG

KmPlot

Easy to use functions plotter. Also from the edu.kde.org project.

KmPlot

Octave

Octave is a scripting language with which you can numerically solve mathematical problems.

The program can be used to for example to solve differential equations or for matrix calculations.

Octave is almost completely compatible to the commercial program MATLAB.

Octave

OpenOffice.org Calc

Calc the spreadsheet editor of OpenOffice.org: With Calc you can analyse, sum up and present your data. Diagrams and tools help evaluate the transparency of your numbers. A fully integrated help-system makes inputing complex formulas easier. Add external data using the data-pilot, sort and filter it, create subtotals and statistical analysis. Use the preview to find the suitable representation out of 13 categories of 2-D and 3-D graphics including line-, surface-, bar-, pie-, XY- and net-charts with dozens of variations.

OpenOffice.org

Qalculate

The program Qalculate is not simply a calculator on you computer but a mathematical and technical calculator with many functions:

Due to that is Qalculate perfect for checking your own calculations or to shorten already proven methods of calculating. Besides the normal calculator function Qalculate also offers:

  • Support of complex numbers
  • the mathematical expressions and calculations can contain numbers (also rational numbers), functions, variables (also as words), vectors and matrices
  • all number systems from two (binary numbers) up to 36 Base numbers, time format and roman numerals are possible
  • exact or approximated results can be outputed
  • Differenciation and integration
  • solving equations
  • more than one solution can be given
  • many statistical, financial, geometric and more functions (nearly 200) are available
  • Qalculate has mathematical constants (Pi, e) and many physical constants and elements.
  • via the program gnuplot, draw function and data graphs which you can then save as .png or Postscript files
  • the mathematical expressions can be created, edited and saved as standard XML files
  • Qalculate also has a calculations history so that you can lookup the way you solved a problem

Qalculate

RKWard

RKWard is a graphical user interface for the scripting language R.

R offers a wide variety of functions to solve statistical problems. Furthermore you can expand R by adding extra packages.

RKWard

TuxMath

Tuxmath is a mathematics trainer for children.

The game concept is similar to that of Missile Command. Equations fall from the sky and Tux has to give the right solution to his laser or else the cities will be destroyed.

TuxMath

wxMaxima

Graphical interface for the maxima computer algebra system (CAS). Maxima is an Open-Source and a productive computer algebra system. Some functions and capabilities:

  • Solving equations and systems of equations
  • Simplification and factorization of terms
  • Determination of limits
  • Trigonometrical calculations
  • Manipulation of polynomials
  • Differentiating and solving differential equations of degree 1. and 2.
  • Integrating
  • Matrices
  • various applications of probability calculations

and much more.
(wx)Maxima can represent graphically, 2 and 3 dimensional function graphs and can also print them. It also works with the function plotting program Gnuplot.

wxMaxima
