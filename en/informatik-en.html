  Computer Science Section

Notes

Naturally a lot of programs that work with and under Linux come from the field of computer science, since we are firstly talking here about an Operating System. As a consequence a lot basic programing languages appear here.

Available are compilers for various programing languages: The GNU Compiler Collection (GCC) contains compilers for C, C++, Objective C, Java, Ada, Fortran, Pascal. Also available is Freepascal/Lazarus, a programing environment similar and compatible to Delphi速. Added to the above are scripting languages like for example Perl or python.

Of course there was not enough space on the CD for an integrated development environment (for example Eclipse) for these languages to be used in a graphical environment (for example KDE or Gnome).

Because of that reason the Seminarix-CD only contains the languages Perl and python and they CAN NOT be run from the Seminarix KDE-Menu but only from the command line.

In all cases in which we found additional help on the Internet, we have linked it with Info under the program description (some of it in English only).

Some of them are magazine articles, some are Wiki-Articles. Since we are not the owners of these Articles, we can not guarantee that these links will work when you depend on them to get help.

At the time of writing, all information is current.

In matters of legality, what is written in the Impressum holds.

Programs Introduction

The programs can be started/installed with the help of the program-starter. The programs of this page can be found under the menu item "Computer Science". You can find the program-starter in the "Kde-Menu" in the sub-menu "Utilities", or you can just click on the program-starter icon on the desktop.

KTurtle

Programing with LOGO for beginners in an easy to use programing environment. The project is part of the KDE-EDU project.

KTurtle

Kivio

Kivio is a program for creating flow charts and editing statistics with a style similar to Visio速. Part of KOffice.

Kivio

Quanta Plus

Quanta Plus is a development environment for creation of webpages. It offers project management and supports arbitrary markuplanguages but it focuses on web-based languages and technologies like HTML, XHTML, CSS, JavaScript, Java or PHP.

Quanta Plus supports a multitude of external komponents and tools, which support the development. Therefore Quanta provides a rich but still easy to use environment with which you can create and manage your websites fast and with efficiency.

Quanta Plus

Gambas

Gambas is a very productive development environment with a Basic-dialect that is reminiscent of VisualBasic.

Gambas

Lazarus

Lazarus is a programing environment, with a good set of visual tools like 'object inspector and formula designer', built upon the Pascal-Dialect, freepascal and the graphical library LCL. With Lazarus there is the possibility to develop programs which reach over to other platforms fast and efficiently (the same code can be compiled for various platforms). Lazarus can be seen as an open source alternative to Delphi since Delphi-programs can be usually ported without much effort.

Lazarus

Dia

Dia has similarities to the proprietary program 'Visio'. With Dia various kinds of diagrams, flow charts and structural diagrams can be prepared. Logical switches and networks are also representable. The products can be exported into various formats like EPS, SVG, XFIG, WMF and PNG.

Dia
