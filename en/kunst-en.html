Seminarix is a Live-medium based on sidux, which is developed for the education of teachers.

  Art Section

Notes

In this section under the term "art" you will find sketch and drawing programs which should indirectly be thought as belonging to the handicrafts side of "art".

The question, in which class do graphics programs belong to, had us at first think of the Art Class. The question is though, how much should a graphic program be able to manipulate a file, which makes this classification hard to assess.

Programs Introduction

The programs can be started/installed with the help of the program-starter. The programs of this page can be found under the menu item "Art". You can find the program-starter in the "Kde-Menu" in the sub-menu "Utilities", or you can just click on the program-starter icon on the desktop.

Blender

Blender is an open-source 3D-Graphics program. With Blender you can model three dimensional object, you can texture them (cover them with a specific material), animate them and render them.

Blender offers a built-in video editor and a player-engine.

In the last few years the program was dynamically developed and new functions were added like for example the possibility to simulate liquids and compositing.

Blender

Digikam

With Digikam you can copy pictures from a digital camera to the hard disk.

It offers numerous and powerful picture editing possibilities. If you want to work with the program on a live-DVD, you have to of course mount a writable medium to store the pictures on.

Digikam

Gimp

With Gimp you have an exceptionally powerful program for editing and manipulating graphics files. With its support for multiple layers, numerous filters and a lot of possibilities to edit pictures, it can be compared to expensive and proprietary professional products.

It is also available for other Operating Systems.

Gimp  |  Info Gimp

Gwenview

An ideal picture viewer to search and display a picture collection under KDE4. It is possible to view the pictures in a fullscreen slideshow and to make minor corrections. For example you can rotate pictures or crop them.

Gwenview

ImageMagick

ImageMagick is an open source tool to edit and create raster images. At this time it can read, manipulate and write more than 100 picture formats.

The software package gets also used within web applications to automatically generate pictures and to dynamicaly adapt them for various purposes.

ImageMagick

Inkscape

Inkscape is an Open-Source vector-graphics editor. Its abilities are comparable to those of Illustrator, Freehand, CorelDraw or Xara.

Inkscape uses the SVG file format (Scalable Vector Graphics). Among others, Inskape makes possible the working with forms, paths, text, marking, clones, transparency effects (alpha), transformations, patterns and groups.

Furthermore Inkscape supports Creative-Commons-Metafiles, node editing, layers, complex path operations, Bitmap-tracing, text belonging to a path, text around an object, direct XML editing and much more. Formats like Postscript, EPS, JPEG, PNG and TIFF can be imported. PNG and various vector-based formats can be exported.

It is also available for other Operating Systems.

Info Inkscape

Kdenlive

Kdenlive is a complete solution for video editing based on the newest video technologies and it supports, for example HDV- and AVCHD-Camcorder. Formats like Mpeg2, mpeg4 and h264 AVCHD can be edited. Exporting in any FFMPEG supported format is also possible. Some further features:

  • Multitrack editing with time bar and a theoretical unlimited number of video and audio tracks
  • Tools, to split audio and video from a clip into different tracks, are also available
  • Simple but very powerful video effects
  • Automatic creation of slideshows from folders with pictures with optional blending of the pictures

Kdenlive

Kino

Kino is an Open-Source Video Editor, which allows, editing, cutting, shortening, clips addition and more, to a video which is played on a video camera which is connected via Firewire.

Kino

KSnapshot

With KSnapshot you can easily take screen-shots of your desktop.

This can either be done automatically with time delay or with a mouse click by the user. Depending on your choice, KSnapshot takes a screen-shot of your whole desktop or only a part of it and supports most file formats.

KSnapshot

LiVES

LiVES is a free, open source video editing and VJ tool. Videos can be cut, edited with effects and converted into other formats. LiVES is especially interesting for Visual Jockeys due to the availability of real-time effects like blending. Via an RFX-Plugin-Interface users can extend and adjust LiVES so it meets their needs.

In 2002 was LiVES initially went by the name "Linux Video Editing System“. It now also works under Windows and Mac OS X and because of that renamed to "LiVES is a Video Editing System“.

LiVES

OpenOffice.org Draw

With Draw you can create anything from simple diagrams to dynamic 3-D illustrations with special effects.

OpenOffice.org

recordMyDesktop

With the help of RecordMyDesktop you can create Screencasts: You record the desktop and all that happens on it to a film. In case you want to you can also add sound and then save the film using the free OGG-format. With this you can for example create tutorials explaining how to use some software and use it in a computer science course.

recordMyDesktop

Stopmotion

Video- and Animations program, with which you can create Stop-motion scenes. An example (GPL) can be found in the /beispiele (/examples) directory.

StopMotion

TuxPaint

TuxPaint is an open source drawing program for kids that are 3 years old and above.

It provides easily accessible tools and also funny sound-effects. A cartoon-man introduces you to drawing. Various tools help you draw with creativity.

Tuxpaint  |  Info TuxPaint
