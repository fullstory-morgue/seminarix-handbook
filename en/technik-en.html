Notes

What was said about mathematics also holds here: The computers have, in the area of engineering, acquired a position of great significance. Because of that you can find many such programs for Linux. The variety of platform independent programs, is naturally, longer than those listed here.

Programs Introduction

The programs can be started/installed with the help of the program-starter. The programs of this page can be found under the menu item "Engineering". You can find the program-starter in the "Kde-Menu" in the sub-menu "Utilities", or you can just click on the program-starter icon on the desktop.

KLogic

KLogic is an application for easily building and simulating digital circuits. KLogic allows you to easily build circuits using standard components like AND, OR, XOR and flipflops like RS and JK. To build more complex and reusable circuits, you can create, store and reuse sub-circuits.

KLogic  |   Info Klogic

KSimus

KSimus is a logical circuits simulator (we could also say digital circuits). It has no built-in elements but it has a plug-ins interface with which you can load dynamic libraries which implement the digital elements. Two libraries are provided by the author. »Boolean« implements a RS-Flipflop, AND, OR, XOR and their negation, a pulse generator and an input/output. »Data Recorder« is used for recording signals.

KSimus

KTechlab

KTechlab is a developer's tool for electrical circuits and micro controllers.

The program can simulate many discrete components (logical, linear, integrating, non-linear and reactive).

The simulation and error correction of PIC (Programmable Intelligent Computer) micro-controllers is carried out by gpsim (a multi-functional software simulator for PIC micro-controllers by Microchip).

KTechlab is based on the higher programming systems Flowcode (for the creating of program flowcharts) and Microbe (a basic-like programming language).

Created programs can be applied onto a specific circuit design (a 'virtual PIC') so that more complex micro-controller based circuits can be created.

KTechlab

LabPlot

With LabPlot you can create tables with measured values and then display them as function diagrams. It is possible to create 2D and 3D diagrams. A simple editing of the plotted function eases the adjustment to the users requirements.

The created function plots can be analysed and evaluated. For that LabPlot offers possibilities which are much more advanced than the capabilities of a spreadsheet editor. With LabPlot the student possesses a valuable tool for evaluating the data they get in school experiments, especially students in higher classes.

Some of LabPlot's abilities:

  • Creation, from functions and data files, of surface, polar and ternary plots and pie charts
  • Flexible 3D plot with rotation etc. multiple plots on one work sheet, speed mode for bigger data and data mode to search for single data points
  • Graph cloning and plots deleting/cloning
  • Versatile tables for data manipulation
  • Online plot scaling and editing
  • Evaluation printouts and direct data editing
  • Free or pan zooming. Masking of data points and markers
  • Support for panel-plots
  • Versatile error bar

Function and data analysis:

  • Averaging, smoothing and reduction of data
  • Compression, periodical and time analysis
  • Find spikes
  • Interpolation (Splines, etc.)
  • Varieties
  • Integration
  • Histogram
  • Regression (up-to an order of 10)
  • Non-linear adjustment (also user defined functions with up-to 9 parameters)
  • Fourier-, Laplace, Hankel and Wavelet Transformation (un)folding
  • Image editing
  • Noise, signal filters, and auto / cross-correlation
  • Capability Analysis
  • R for additional functions and descriptive statistics / 1-/2-Sample Tests
  • Support of LaTeX and RichText labels

LabPlot offers import/export filters for more than 80 file formats like ps, eps and pdf. The support for connecting to a data bank like MySql and Postgresql is realised via KexiDB.

LabPlot

PCB

Whoever wants to avoid using ready made circuit boards and wants to design his own instead, he can create them with the help of this program. Available electronic components have to be connected and positioned by the user (integrated is a virtual circuit tester). In addition to automatic conductor evaluation, you can also manually edit those.

PCB

Qcad

Qcad is a program for two dimensional CAD drawing and because of that it is especially suitable for engineering courses.

With qcad you can create technical drawings and plans for buildings, installations, electrical circuits, process schemes, diagrams and mechanical components.

With the help button you can call the manual. This is currently available only in English but Qcad is sufficiently intuitive.

Qcad

Xoscope

Xoscope is a digital oscilloscope, that uses the input from a sound-card's line-in jack. It allows the simultaneous display of 8 signals.

Time scales can be divided into intervals ranging from one microsecond to two seconds.

It contains external mathematical functions like inverse function, sum, difference , mean value, xy and FFT. 23 buffer storages, 5 automatic measurements and it can load and save files.

Warning:You should never connect devices with more than 1,5 volts to your sound-card's line-in jack because this could destroy the sound-card.

By higher voltages you can reduce the voltage to 1,5V by using a voltage divider so that you do not overcharge your sound-card.

Jacks with headphone cables are a suitable measuring cable and can be cheaply bought in many stores. Xoscope is mostly suitable for experiments with time sensitive electrical voltages like for example to measure the generator principal, capacitor's charging and discharging and to measure oscillation processes with electrical signal conversion (for example via a microphone).

To measure D.C. voltage it is necessary to convert and calibrate the voltage by using a so-called VCO component (voltage controlled oscillator) because the line-in input is via an upstream capacitor not "passable" by D.C. voltage.

Xoscope
