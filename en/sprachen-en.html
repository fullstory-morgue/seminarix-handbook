Seminarix is a Live-medium based on aptosid, which is developed for the education of teachers

  Languages Section

Notes

It is the opinion of experts that to learn a language it is necessary to not only have dictionaries and translation helpers but also suitable exercise material.

A few examples can be found here.

Programs Introduction

The programs can be started/installed with the help of the program-starter. The programs of this page can be found under the menu item "Languages". You can find the program-starter in the "Kde-Menu" in the sub-menu "Utilities", or you can just click on the program-starter icon on the desktop.

Ding

Ding is a program to search within a German-English dictionary with approx. 216.000 entries.

You can, in Ding, set the search parameters, the language (German or English) and the color.

It has a history function and a help function. Sensible keyboard and mouse assignments make searching fast and easy.

Ding allows you, in an easy way, to display translation suggestions of words or word groups from various documents (web-pages, text documents, slides, etc.).

For that it is enough to highlight the respective word or word group and the translation will be automatically be displayed.

Ding

Hanzim

Learn Chinese easily (is this even possible?).

Hanzim

Kiten

Kiten includes three main functions:

  • English-Japanise dictionary
  • Kanji dictionary, with multiple ways to look up specific characters
  • Tool to help you learn Kanji

Kiten

KLettres

KLettres is designed to help the user to learn the alphabet in a new language and then to learn to read simple syllables.

The program is suitable for children which are 2 years old or older but also for adults that want to learn the basics of a foreign language.

At this time there are 11 languages available: Czech, Danish, Dutch, English, French, German, Hindi (romanized), Italian, Luganda, Spanish, Slovak. English and French are preinstalled.

KLettres

KWordQuiz

It is a tool that works with flashcards and it helps the user train his vocabulary. It can be used for learning a language or any kind of "special" terminology.

The program allows access to various vocabulary data banks. It supports .kvtml files, which are used by other KDE programs like for example KVocTrain, .wql files, which are used by WORD Quiz for Windows, .cvs files with text separated by commas and .xml .gz files created by the application pauker.

This package is part of the official KDE Edutainment module.

KWordQuiz

lletters

A program, to learn the meaning of numbers and letters, for children in preschool and primary school.

After you click on a number or a letter, a picture of an object with a name that begins with the corresponding letter (for a number the corresponding number of objects) including the correct spelling of the word is shown.

Unfortunately this program is only available in English and French. Because of that it is also suitable for older children which are starting to learn a foreign language.

lletters

Parley

There exist for KDE a rich selection of vocabulary trainers and the most popular ones are called KWordquiz, KVoctrain and Parley.

What all of those have in common is the usage of KVTML vocabulary files. There are many predefined KVTML-files but you can also create your own. This is best done with KWordquiz, since it is easier to do there. It plays no role in which of the programs the file is afterwards used. Parley is the port of KVoctrain for KDE4 and it offers many configurable options.

Many information on the tools and the creation of KVTML-files

Parley
